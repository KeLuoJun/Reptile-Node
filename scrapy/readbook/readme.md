# 框架使用说明

该框架适用范围广，但在使用时需要根据具体需求进行以下修改：

---

## 1. `read.py` 文件
- **爬取 URL**  
  修改爬取的目标 URL。
  
- **XPath 解析路径**  
  根据目标网页的结构，调整 XPath 解析路径。

---

## 2. `pipelines.py` 文件
- **保存路径**  
  修改数据保存的路径，确保符合项目需求。
  
- **SQL 语句**  
  根据数据库表结构，调整 SQL 插入或更新语句。

---

## 3. `items.py` 文件
- **数据类型**  
  定义或修改数据字段的数据类型，确保与爬取内容一致。

---

## 4. `settings.py` 文件
- **数据库配置**  
  配置数据库连接信息，包括：
  - 数据库地址
  - 端口号
  - 用户名和密码
  - 数据库名称
- **robots协议**
  配置robots协议，确保爬虫遵守robots协议。
- **ITEM_PIPELINES配置**
  配置ITEM_PIPELINES，确保数据处理按照指定的管道进行处理。
- **LOG_FIL**
  配置日志文件路径，确保日志记录符合项目需求。

---

完成上述修改后，即可运行框架进行数据爬取和处理。
